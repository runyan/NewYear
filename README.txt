制作贺年卡并分享给微信好友，微信朋友圈以及新浪微博
安卓APP 使用Android Studio开发 
基于慕课网 Android-心愿分享课程 http://www.imooc.com/learn/592 拓展
使用
    微博分享SDK http://open.weibo.com/
    微信分享SDK https://open.weixin.qq.com/
    开源组件：
            android-gif-drawable:https://github.com/koral--/android-gif-drawable
            material_dialogs:https://github.com/afollestad/material-dialogs
    注意：
	1.使用分享SDK前需要去网站注册成为开发者，获得app_id，app_key后才可以使用分享功能。
	2.由于使用分享功能需要应用签名，在测试时可能无法使用分享功能，打包生成signed apk后可正常使用
	3.使用新浪微博的分享功能时，需要将应用在某个应用商店后上架，否则只可以使用开发者或者测试组用户的微博账号进行分享。
